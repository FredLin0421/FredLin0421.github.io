Recommendation Systems

Dec 2023 — Robustness in HW environments

Project Summary

This project evaluates the robustness of recommendation systems in large-scale hardware environments. It investigates the impact of hardware faults on system performance and proposes strategies for fault detection and mitigation.

My Role

Researcher: conducted experiments, analyzed system behavior under faults, and contributed to the development of robust algorithms.

Technologies Used

  • Python, TensorFlow
  • Recommendation Algorithms
  • Fault Injection, System Analysis

Details

The project involved simulating hardware faults in recommendation system pipelines and measuring their effect on recommendation accuracy and user experience. The findings led to the development of new monitoring and correction techniques for production systems.
